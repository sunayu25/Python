# Lesson 3 序列 Sequence

## ⬆️什麼是序列 Sequence

程式在執行時，會有由左而右，由上而下的執行順序，所以後面執行的程式可能會對前面的結果造成覆蓋。

```python
x = 10
x = 20
print(x)
```
