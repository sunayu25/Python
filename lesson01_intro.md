## 📖 Lesson 1: 輸入與輸出 Input & Output


我們可以把程式想像成一個機器，在寫程式的過程，我們就是在設計一台機器：
![alt text](image-9.png)

`[ 輸入 input ]` ──> `[ 處理 process ]` ──> `[ 輸出 output ]`

### 📥 輸出函式 `print()`
用來將程式的結果顯示在螢幕上。

![alt text](image-11.png)

#### (1) 輸出文字
*   若要輸出一段文字，我們需要先標註這段內容是「字串（文字）」。
*   使用**英文雙引號 `""`** 即可標註文字。
*   **實作練習：** 使用 `print()` 輸出 `Hello World`

![alt text](image-10.png)

```python
    print("Hello World")
```
#### 試試看： 如果是中文 “哈囉妳好” 怎麼輸出？

#### (2) 輸出數字
*   數字不需加上引號，直接寫入括號內即可。
*   **實作練習：** 使用 `print()` 輸出 `123`
```python
    print(123)
```
#### 試試看： 輸出 1-10 中的偶數數字 怎麼輸出？

#### (3) 輸出算術結果
*   可以讓 Python 幫你計算數學題並直接輸出結果。
*   **實作練習：** 計算並輸出 123 + 456
```python
    print(123 + 456)
```
#### 試試看： 請程式幫你計算出 123+492+572-234-255+245 =？

#### (4) 輸出多個內容
*   Python 可以使用一個 `print()` 輸出複數個內容，只需在各個輸出內容之間使用**逗號 `,`** 隔開。
*   **實作練習：** 使用 `print()` 一行輸出 `123` 與 `早安`
```python
    print(123, "早安")
```
#### 試試看： 輸出 你的中文名字, 英文名字, 今年幾歲？

#### (5) 分隔符號 `sep`
*   我們發現兩個輸出內容之間預設會被「空格」隔開，原因是 `print()` 的預設分隔符號（separator）是空格。
*   我們可以手動更改分隔符號，只需使用 `sep` 參數即可。
*   **實作練習：** 輸出 1~5，並指定中間使用逗號 `,` 隔開
```python
    print(1, 2, 3, 4, 5, sep=",")
```

### 📤 輸入函式 `input()`
使用輸入函式可以讓程式獲得使用者從鍵盤輸入的內容。

*   語法：`input('提示字元')`
*   提示字元用來要求使用者輸入指定內容，例如："請輸入帳號"。
*   **實作練習：** 輸入一串文字並讓程式原封不動輸出該文字：
```python
    print(input("請輸入一段文字："))
```

#### 💡 進階技巧：`split()` 與 `end` 參數
*   **`split()` 分割功能：** 若我們獲取的同一行輸入中包含多個資訊，可以使用 `.split()` 將其分割，預設會以「空格」作為分割點。
```python
input().split()  將輸入內容從空格分割
```
*   **`end` 結尾參數：** `print()` 輸出最後預設會自動加上一個「換行符號」。我們可以使用 `end` 參數來指定結尾符號，讓下一次的輸出接在後面而不換行。
```python
print(輸出內容,end='結尾符號')
```
